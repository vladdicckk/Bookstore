# Bookstore
«Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра (ВСП «ППФК НТУ «ХПІ»).
Відповідно до поставленого завдання необхідно розробити онлайн магазин книг з передбаченням двох типів облікових записів – власника книжкового міні-магазину(книгарні) та користувача(покупця книг).
Перший тип облікового запису – власник книгарні, де він має можливість обміняти чи продати непотрібні книги, він повинен мати можливість:
	мати окремий екран з блоками книг: рекомендовані, останні додані та рандомні книги рандомного жанру;
	додавати книги в блок «рекомендовані»;
	додавати, видаляти, редагувати інформацію про книги;
	змінювати статус наявності книги;
	редагувати інформацію профілю;
	мати можливість переглядати заявки на купівлю/обмін книг;
	мати можливість переглядати інформацію в профілі інших користувачів.
 Покупець повинен мати особистий профіль. Профілі власників книгарні і їх книги можна переглянути і без реєстрації, але для купівлі чи обміну є доступ лише в авторизованих покупців. Користувач матиме можливість:
	шукати міні-магазини за фільтрами;
	шукати необхідні книги за фільтрами;
	відправити запит бібліотекарю на обмін чи купівлю книги;
	змінювати інформацію профілю;
	переглядати профілі власників книгарень;
	переглядати детальну інформацію про книги.
